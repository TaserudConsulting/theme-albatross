# Hugo theme: Albatross

This is a hugo theme. It's quite flexible.

## Features

- Responsive
- Customizable colors
- Dark and light mode
- Blog support
- Multi language support
- Support for resturant style menu pages
- Support for 3d models with an embedded viewer on listing page
